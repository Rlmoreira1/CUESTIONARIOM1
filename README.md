# Sistema de Evaluación Interactiva

Aplicación web interactiva desarrollada con React + Vite para evaluar conceptos en alumnos.

## Características
- 30 preguntas aleatorias por alumno.
- 3 tipos de preguntas: Opción múltiple, completar texto y SÍ/NO.
- Calificación automática inmediata.
- Exportación de resultados a CSV (botón en la pantalla de inicio).
- Diseño premium responsivo.

## Desarrollo Local
1. Instalar dependencias: 
pm install
2. Correr servidor de desarrollo: 
pm run dev

## Despliegue
Diseñado para ser desplegado en plataformas como Lovable, Vercel o Netlify conectando este repositorio de GitHub.
